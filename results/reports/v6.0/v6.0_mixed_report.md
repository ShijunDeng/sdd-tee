### 9.4 总结

在总 Token 数不变的假设下：

- 全部使用 glm-5 的成本为 **$53.22**
- 当前混搭配置的成本为 **$35.00**
- 混搭配置节省 **$18.22**
- 成本降低约 **34.2%**

因此，在多 Agent 场景中，将规划、任务拆解、代码实现、文档等阶段切换到 MiniMax-M2.5，并将验证阶段切换到 glm-4.7，可以在不改变总 Token 消耗的前提下，将整体成本从 **$53.22** 降至 **$35.00**。
