This section is for **contributors** and **platform engineers** extending AgentCube or building custom agent runtimes on top of it.

## What you will find here

- How the repository is organized and where controllers, APIs, and SDKs live
- How to **build custom runtimes** compatible with `AgentRuntime` and `CodeInterpreter`
- Local development workflows for Go services, Python CLI, and the SDK
- Testing expectations for unit, lint, and end-to-end suites
